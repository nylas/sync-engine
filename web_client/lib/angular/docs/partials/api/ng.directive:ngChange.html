ngChange
directive in module ng

Description

Evaluate given expression when user changes the input. The expression is not evaluated when the value change is coming from the model.

Note, this directive requires ngModel to be present.

Usage

as attribute
<input ng-change="{expression}">
   ...
</input>

Parameters

ParamTypeDetails
ngChangeexpression

Expression to evaluate upon change in input value.
