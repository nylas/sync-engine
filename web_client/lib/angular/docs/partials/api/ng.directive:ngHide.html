ngHide (directive in module ng )

Improve this doc

Description

The ngHide and ngShow directives hide or show a portion of the DOM tree (HTML) conditionally.

Usage

as attribute
<ANY ng-hide="{expression}">
   ...
</ANY>
as class
<ANY class="ng-hide: {expression};">
   ...
</ANY>

Parameters

  • ngHide – {expression} –

    If the expression is truthy then the element is shown or hidden respectively.
